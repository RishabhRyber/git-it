# git-it
Command line tool to get info about any github user or repositories
     
```
       _ _             _ _
  __ _(_) |_          (_) |_
 / _` | | __|  _____  | | __|
| (_| | | |_  |_____| | | |_
 \__, |_|\__|         |_|\__|
 |___/

```


  Usages: python git-it [options] [parameter]

     Options:
          -u, -U
             Use the github user-id is provided
        -e, -E
     
             Use the email id
        -r, -R
             Show repository details also
